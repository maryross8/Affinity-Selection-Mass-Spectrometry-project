# Affinity Selection-Mass Spectrometry (ASMS) project

The synthesis of libraries is lengthy, especially when extensive purification is required. When hit discovery is the aim we want to be able to test lots of molecules, fast. In an academic lab setting however, this is lengthy and laborious and time is often spent synthesising molecules with low binding affinity, rendering them useless and the time spent synthesising essentially wasted. Avoiding lengthy purification procedures is a way to cut down on labour and time, shifting the focus from making molecules to testing them, which is the ultimate goal. 

The aim of this project is to validate a methodology for the use of crude product libraries in ASMS to accelerate the discovery of hit molecules. This involves the synthesis of an amide library for screening against the WDR91 protein, which can be found [here](https://github.com/maryross8/Affinity-Selection-Mass-Spectrometry-project/wiki/Grid-Design). The pure compounds will be synthesised and screened against WDR91 to give an affinity ranking. Among these molecules is a best binder identified by Kangping Liu (Todd group, UCL). Following this, the library will be remade without purification, giving the crude product library. This will then be screened against WDR91 to replicate the affinity ranking thus validating a methodology for the use of crude product libraries to accelerate the identification of hits.
