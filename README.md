# Affinity Selection-Mass Spectrometry (ASMS) project

A project within the Todd group at UCL devoted to investgating the use of crude product libraries in ASMS to accelerate the identification of hits for target proteins. The project involves the synthesis of an amide library for screening against the WDR91 protein, which can be found [here](https://github.com/maryross8/Affinity-Selection-Mass-Spectrometry-project/wiki/Grid-Design).
